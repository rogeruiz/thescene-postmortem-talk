The Scene postmortem or: How we learned to stop worrying & love the Runloop.

A talk in three parts.

We started in February of 2014 on Ember, 1.4.0.

We set out to learn Ember "the right way".

"The Ember way"

The Scene is built on Rails serving up the Ember application.

Reasons for going with Rails
  • Build what we know
  • Code reuse from similar projects
  • Prebaking intial payload on load
  • No Ember App Kit nor Ember-CLI
  • Discourse as a solid example

We launched on July 15th, 2014 on Ember, 1.7.0.beta.canary1
